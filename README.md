# AI Services Agency

AI Services Agency is a Streamlit web app that uses OpenRouter models to generate business and technical project analysis reports.

A user enters a project idea, and the app produces an agency-style report with input from five virtual roles:

- CEO
- CTO
- Product Manager
- Lead Developer
- Client Success Manager

The app can also test and rank multiple OpenRouter models to find the best working model before generating a full report.
